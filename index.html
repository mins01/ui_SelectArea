SelectArea

영역 지정 UI

  • jQuery나 bootstrap 등이 없이
  • 최대한 Tag(node)의 기본 기능을 쉽게 사용하도록
주요 멤버 변수
  • SelectArea.selectedArea : 선택된 영역을 나타내는 DIV. CSS .selectArea-layout 참고
  • SelectArea.outOfRange : 영역 기준 대상을 벗어날 수 있는지에 대한 설정(default:false)
주요 멤버 변수
  • SelectArea.target : node. 이 노드 위에 SelectArea 가 보여진다.
  • SelectArea.selectedArea : 선택된 영역을 나타내는 DIV. CSS .selectArea-layout 참고
