SelectArea

영역 지정 UI

  • jQuery나 bootstrap 등이 없이. 최대한 Tag(node)의 기본 기능을 사용하도록.
  • github : https://github.com/mins01/ui_SelectArea
주요 멤버 변수
  • SelectArea.selectedArea : 선택된 영역을 나타내는 DIV. CSS .selectArea-layout 참고
  • SelectArea.outOfRange : 영역 기준 대상을 벗어날 수 있는지에 대한 설정(default:false)
주요 멤버 변수
  • SelectArea.setTarget //위치 기준 대상
  • SelectArea.setRangeTarget //이동 위치 기준 대상
  • SelectArea.show
  • SelectArea.hide
  • SelectArea.getSelectedAreaRect
  • SelectArea.moveAndSize
  • SelectArea.moveBy
  • SelectArea.sizeBy
  • SelectArea.resync
  • SelectArea.syncPosCoordinateBy
  • SelectArea.sizeBy
CustomEvent
  • change : selectedArea 가 변경되면 발생한다
